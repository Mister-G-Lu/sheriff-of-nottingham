# Coverage Tools

Test coverage analysis tools for Sheriff of Nottingham.

## Current Coverage: 48%

## Tools

### analyze_coverage.py
Comprehensive coverage analysis with detailed reporting.

```bash
cd coverage_tools
python analyze_coverage.py
```

### run_tests_with_coverage.py
Run all tests and generate coverage data.

```bash
cd coverage_tools
python run_tests_with_coverage.py
```

### .coveragerc
Coverage configuration file (used automatically by coverage tool).

## Quick Commands

```bash
# From project root
cd coverage_tools

# Run analysis
python analyze_coverage.py

# Run tests
python run_tests_with_coverage.py

# Generate HTML report
cd ..
coverage html
open coverage_html_report/index.html
```
